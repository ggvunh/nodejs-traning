# Nodejs validation package

[![Build Status](https://travis-ci.org/ggvunh/nodejs-validation.svg?branch=master)](https://travis-ci.org/ggvunh/nodejs-validation)

## Requirement
- Node 6

## Install

Via Npm

``` bash
$ npm install --save
```

## Usage
Run example to show results from log:

```
npm start
```
## Testing

``` bash
$ mocha test
```
